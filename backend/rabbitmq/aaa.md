# RabbitMQ 消息队列

欢迎来到 RabbitMQ 学习专

RabbitMQ 是一个开源的消息代理软件，它实现了高级消息队列协议（AMQP）。

## 主要特性

- **可靠性**: 消息持久化，集群，高可用
- **灵活的路由**: 支持多种交换机类型
- **多种协议**: 支持AMQP, STOMP, MQTT等
- **管理界面**: 提供Web管理界面

## 学习路径

1. 基础概念和安装
2. 消息发送和接收
3. 交换机和队列
4. 高级特性和最佳实践
